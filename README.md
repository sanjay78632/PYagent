# PyAgent

An autonomous AI agent built from scratch. Starting as a simple chatbot with memory, evolving into a tool-using agent that can browse, read files, execute code, and take autonomous actions.

## Vision (6-Month Roadmap)

**Month 1**: Conversation engine + persistent memory  
**Month 2**: Web search and real-time information retrieval  
**Month 3**: FastAPI backend + REST API integration  
**Month 4**: Code execution sandbox + autonomous task handling  
**Month 5**: Multi-tool orchestration + agentic decision making  
**Month 6**: Production-ready agent with proper error handling and monitoring  

## Current Status: Month 1, Week 1, Day 1

Building the core conversation loop and memory system.

## Tech Stack

- Python 3.12+
- Anthropic Claude API (coming Week 3)
- Rich library (for terminal UI)
- Requests library (for HTTP)

## Installation

```bash
git clone https://github.com/sanjay78632/PYagent.git
cd PYagent
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

The agent will start. Type messages to interact. Type `exit` to quit.
